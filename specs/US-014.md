## Key v4 Features

### Multi-Driver System (NEW)

- Local storage
- S3 storage
- Extensible for other providers

### Storage Manager

```typescript
import { storage } from "@warlock.js/core";

// Upload file
await storage.put("path/to/file.jpg", buffer);

// Get file
const file = await storage.get("path/to/file.jpg");

// Delete file
await storage.delete("path/to/file.jpg");

// Generate URL
const url = storage.url("path/to/file.jpg");
```

### 2. Configuration

```typescript
// src/config/storage.ts
export default {
  default: "local", // or "s3"

  disks: {
    local: {
      driver: "local",
      root: "storage",
    },
    s3: {
      driver: "s3",
      bucket: env("S3_BUCKET"),
      region: env("S3_REGION"),
      key: env("S3_KEY"),
      secret: env("S3_SECRET"),
    },
  },
};
```
