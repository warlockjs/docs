## Content Outline

### 1. Prerequisites

- Node.js 20+
- Package manager (Yarn/NPM/Pnpm)
- Database: MongoDB or PostgreSQL

### 2. Create New Project

```bash
yarn create warlock
# or
npx create-warlock
# or
pnpm create warlock
```

### 3. What Gets Created

- Project structure overview
- Key files (warlock.config.ts, .env)

### 4. Start Development Server

```bash
yarn dev
```

### 5. What to Expect

- File processing message
- Health checks starting
- Server ready message
- Hot reload enabled

### 6. Tooling

- Generator Z VSCode extension
