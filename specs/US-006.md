### 1. Environment Variables (.env)

### 2. Environment File Types

- .env (development default)
- .env.production
- .env.shared (common variables)

### 3. Default .env Variables

```env
APP_NAME=myapp
PORT=3000
HOST=127.0.0.1

# Database (MongoDB)
DB_HOST=127.0.0.1
DB_PORT=27017
DB_NAME=myapp

# OR Database (PostgreSQL)
PG_HOST=127.0.0.1
PG_PORT=5432
PG_NAME=myapp
```

### 4. Configuration Files (src/config/)

- app.ts
- database.ts
- http.ts
- auth.ts
- mail.ts
- cache.ts
- validation.ts
- uploads.ts

### 5. The Pattern

```
.env (secrets) → config (logic) → app (usage)
```

### 6. Example: Database Config

```typescript
// src/config/database.ts
export default {
  driver: env("DB_DRIVER", "mongodb"),
  host: env("DB_HOST"),
  port: env("DB_PORT"),
  name: env("DB_NAME"),
};
```
