## Pages

### 1. Image Processing

```typescript
import { Image } from "@warlock.js/core";

// Resize
const resized = await Image.resize(file, {
  width: 300,
  height: 300,
  fit: "cover",
});

// Compress
const compressed = await Image.compress(file, {
  quality: 80,
});

// Watermark
const watermarked = await Image.watermark(file, {
  image: "watermark.png",
  position: "bottom-right",
});

// Format conversion
const webp = await Image.convert(file, "webp");
```
