## Error Handling

:::warning Common Errors
**Error: Feature not initialized**
- Cause: Calling methods before `initialize()`
- Solution: Always call `await feature.initialize()` first

**Error: Invalid option value**
- Cause: Passing incorrect type to configuration
- Solution: Check the Configuration Options table for correct types
:::

## Best Practices

1. **Always initialize first** - Call `initialize()` before using any methods
2. **Use TypeScript types** - Import and use provided types for better IDE support
3. **Handle errors gracefully** - Wrap calls in try-catch blocks
