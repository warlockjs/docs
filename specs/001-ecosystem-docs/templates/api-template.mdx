## Methods

### initialize()

```ts
async initialize(): Promise<void>
```

Initializes the instance. Must be called before using other methods.

**Returns:** `Promise<void>`

**Throws:** `InitializationError` if already initialized

**Example:**

```ts title="src/initialize-example.ts"
import { ClassName } from "@warlock.js/package";

const instance = new ClassName({ option1: "value" });
await instance.initialize();
// Instance is now ready to use
```

### get()

```ts
async get(id: string): Promise<DataType | null>
```

Retrieves data by its identifier.

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | The unique identifier |

**Returns:** `Promise<DataType | null>` - The data if found, null otherwise

**Example:**

```ts title="src/get-example.ts"
import { ClassName } from "@warlock.js/package";

const instance = new ClassName({ option1: "value" });
await instance.initialize();

const data = await instance.get("abc123");
if (data) {
  console.log(data);
  // Output: { id: "abc123", name: "Example", ... }
}
```

### create()

```ts
async create(data: CreateInput): Promise<DataType>
```

Creates a new record with the provided data.

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `data` | `CreateInput` | Yes | The data to create |

**Returns:** `Promise<DataType>` - The created record

**Throws:** `ValidationError` if data is invalid

**Example:**

```ts title="src/create-example.ts"
import { ClassName } from "@warlock.js/package";

const instance = new ClassName({ option1: "value" });
await instance.initialize();

const created = await instance.create({
  name: "New Record",
  email: "example@email.com",
});

console.log(created);
// Output: { id: "generated-id", name: "New Record", email: "example@email.com", createdAt: "2026-01-15T..." }
```

## Events

### created

Fired when a new record is created.

```ts
instance.on("created", (data: DataType) => {
  console.log("New record created:", data);
});
```

**Payload:** `DataType` - The created record

### updated

Fired when a record is updated.

```ts
instance.on("updated", (data: { old: DataType; new: DataType }) => {
  console.log("Record updated from", data.old, "to", data.new);
});
```

**Payload:** `{ old: DataType; new: DataType }` - The old and new record data
