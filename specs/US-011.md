## Key v4 Changes

### Route Builder (NEW)

```typescript
router
  .route("/posts")
  .get(listPostsController)
  .post(createPostController)
  .nest("/:id", (builder) => {
    builder
      .get(getPostController)
      .put(updatePostController)
      .delete(deletePostController);
  });
```

### Response Streaming (NEW)

```typescript
response.stream(async (stream) => {
  stream.send({ chunk: 1 });
  stream.send({ chunk: 2 });
  stream.end();
});
```

### RESTful Semantic Aliases (NEW)

- list(), create(), show(), update(), destroy()
