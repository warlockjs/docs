## Key Points

### Core Extends Seal

- Seal is standalone package
- Core adds: file validation, database rules
- `v` imported from `@warlock.js/seal`

### Validation Pattern

```typescript
import { type RequestHandler } from "@warlock.js/core";
import { v } from "@warlock.js/seal";

export const myController: RequestHandler = async (request, response) => {
  // request.validated() returns validated data
  const data = request.validated();
  // ...
};

myController.validation = {
  schema: v.object({
    email: v.string().email().required(),
    password: v.string().min(8).required(),
  }),
};
```

### Database Rules (Core adds)

```typescript
v.string().unique("users", "email"); // Check unique in DB
v.string().exists("roles", "id"); // Check exists in DB
```

### File Validation (Core adds)

```typescript
v.file().image().maxSize({ size: 2, unit: "MB" }).saveTo("uploads");
```
