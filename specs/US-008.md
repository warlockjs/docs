## Content Outline

### 1. Introduction

- i18n support built-in
- Module-based translation files
- Fallback locale

### 2. Configuration

```typescript
// src/config/app.ts
export default {
  localeCode: "en",
};
```

### 3. Module Locales File

```typescript
// src/app/users/utils/locales.ts
import { setTranslations } from "@warlock.js/core";

setTranslations("users", {
  en: {
    welcome: "Welcome",
    emailSent: "Email sent successfully",
  },
  ar: {
    welcome: "مرحبا",
    emailSent: "تم إرسال البريد بنجاح",
  },
});
```

### 4. Using Translations

```typescript
import { trans } from "@warlock.js/core";

trans("users.welcome"); // "Welcome" or "مرحبا"
```

### 5. Request Locale

- Header: `Accept-Language`
- Auto-detected by framework
