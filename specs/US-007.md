## Content Outline

### 1. What Are Connectors?

- Manage external service connections
- Database, cache, HTTP, storage, etc.
- Handle lifecycle (connect, disconnect, restart)

### 2. Built-in Connectors

| Connector    | Priority | Purpose                |
| ------------ | -------- | ---------------------- |
| Database     | 1        | Database connection    |
| Cache        | 2        | Redis/cache connection |
| HTTP         | 3        | Fastify server         |
| Storage      | 4        | File storage           |
| Communicator | 5        | Message bus (Herald)   |

### 3. Connector Lifecycle

```
Application Start:
1. Database.init()
2. Cache.init()
3. HTTP.init()
4. Storage.init()
5. Communicator.init()

Application Shutdown (reverse):
1. Communicator.shutdown()
2. Storage.shutdown()
3. HTTP.shutdown()
4. Cache.shutdown()
5. Database.shutdown()
```

### 4. When Connectors Restart

- Config file changes (during dev)
- Triggered by dev server HMR
