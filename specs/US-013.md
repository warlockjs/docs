## Key v4 Features

### Adapter Pattern (NEW)

- Not coupled to Cascade anymore
- Can use Prisma, custom adapters
- Interface-based

### Scope Filters (NEW)

```typescript
// Repository definition
filterBy: {
  active: "scope",  // Uses query.scope("active")
}

// Usage
repository.list({ active: true });
```

### Cursor Pagination (NEW)

```typescript
// Instead of offset-based
repository.list({
  cursor: "lastId",
  limit: 20,
});
```

## Content Outline

### Basic Repository

```typescript
import { RepositoryManager } from "@warlock.js/core";
import { Post } from "../models/post";

class PostsRepository extends RepositoryManager<Post> {
  model = Post;

  filterBy = {
    title: "like",
    status: "=",
    active: "scope",
  };
}

export const postsRepository = new PostsRepository();
```

### Using Repository

```typescript
// List with filters
const posts = await postsRepository.list({
  title: "hello",
  status: "published",
  active: true,
});

// Get single
const post = await postsRepository.find(id);

// Create
const post = await postsRepository.create(data);
```
