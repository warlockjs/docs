## Pages

### 1. Commands Overview

| Command            | Description              |
| ------------------ | ------------------------ |
| `warlock dev`      | Start development server |
| `warlock build`    | Build for production     |
| `warlock generate` | Generate code            |
| `warlock migrate`  | Run migrations           |

```bash
# Development
yarn dev          # or: npx warlock dev

# Production build
yarn build        # or: npx warlock build

# Generate module
yarn generate module users
```

### 2. Generating Modules

- Generate full module
- Generate controller
- Generate model
- Generate repository

```bash
# Full module
warlock generate module posts

# Controller only
warlock generate controller posts/create-post

# Model only
warlock generate model posts/post
```

### 3. Custom Commands

```typescript
// src/commands/my-command.ts
import { command } from "@warlock.js/core";

command("my:command", async (args) => {
  console.log("Running my command with:", args);
});
```
