# 🗺️ Roadmap

Welcome to your Cascade adventure! Here's your guided path to MongoDB mastery, whether you're new or a seasoned dev. 🌟

## 🏷️ Key Terminology

- 🗂️ **Collection**: Like a table in SQL—a group of documents.
- 📄 **Document**: A single record (like a row in SQL), usually a JSON object.
- 🏷️ **Field**: A property of a document (like a column in SQL).
- 🔍 **Query**: A request to the database to do something.
- 🧑‍💻 **Model**: A class that manages a collection's documents.
- 🏗️ **Schema**: The blueprint for a collection—defines fields and types.
- 🧩 **Aggregate**: A pipeline of stages to process documents.
- 🛠️ **CRUD**: Create, Read, Update, Delete—core database actions.
- 📖 **Pagination**: Splitting results into pages for easier browsing.
- 🕵️‍♂️ **Explain**: Get info about how a query runs (for debugging/optimizing).
- 🔄 **Upsert**: Update if exists, otherwise insert a new document.
