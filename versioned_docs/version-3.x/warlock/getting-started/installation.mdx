## Pre-requisites

Warlock.js requires Node.js version <span className="text-primary">20</span> or higher. You can download and install Node.js from the official website [nodejs.org](https://nodejs.org/). Ensure you also check the [MongoDB Requirements](./../../cascade/getting-started/installation.mdx).

## Installation

To start a new Warlock.js project, execute the following command:

<Tabs>
  <TabItem value="Yarn" label="Yarn">

```bash
yarn create warlock
```

  </TabItem>
  <TabItem value="NPM" label="NPM">

```bash
npx create-warlock
```

  </TabItem>
  <TabItem value="Pnpm" label="Pnpm">

```bash
pnpm create warlock
```

  </TabItem>
</Tabs>

Follow the installation instructions, and you're all set! 🎉

## VSCode Extension

Warlock.js modules can be generated using the [Generator Z](https://marketplace.visualstudio.com/items?itemName=hassanzohdy.gnz) VSCode extension. Install it from the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=hassanzohdy.gnz).

## Run the project

Navigate to the project directory and run the following command:

<Tabs>
  <TabItem value="Yarn" label="Yarn">

```bash
yarn start
```

  </TabItem>
  <TabItem value="NPM" label="NPM">

```bash
npm start
```

  </TabItem>
  <TabItem value="Pnpm" label="Pnpm">

```bash
pnpm start
```

  </TabItem>
</Tabs>
