# 🛠️ Installation

Welcome to the first step of your Cascade journey! 🎉

Cascade is part of the [Warlock.js](https://github.com/hassanzohdy/warlock.js) full-stack framework, but you can use it standalone too. Let's get you set up in seconds!

## 📦 Install the Package

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs>
  <TabItem value="NPM" label="NPM" default>

```bash
npm i @warlock.js/cascade
```

  </TabItem>
  <TabItem value="Yarn" label="Yarn">

```bash
yarn add @warlock.js/cascade
```

  </TabItem>
  <TabItem value="Pnpm" label="Pnpm">

```bash
pnpm add @warlock.js/cascade
```

  </TabItem>
</Tabs>

:::info
Cascade requires a database driver peer dependency. Install the one that matches your database:

- **MongoDB:** `npm i mongodb`
- **PostgreSQL:** `npm i pg`
:::

:::tip
We highly recommend using **Typescript** for the best developer experience. All docs and examples use Typescript.
:::
