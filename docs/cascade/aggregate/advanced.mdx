# ⚙️ Advanced Aggregation

Beyond `groupBy`, the QueryBuilder exposes quick scalar aggregation methods and atomic increment/decrement operations.

## 📊 Scalar Aggregation Methods

These terminal methods execute immediately and return a single computed value — no `.get()` needed.

### count()

Count all rows matching the current filters:

```ts title="src/repositories/user.repo.ts"
import { User } from "./user.model";

const total = await User.query().where("active", true).count();
// => 1248
```

### sum(field)

Sum a numeric column across all matching rows:

```ts title="src/repositories/order.repo.ts"
import { Order } from "./order.model";

const revenue = await Order.query()
  .where("status", "paid")
  .sum("amount");
// => 48250.75
```

### avg(field)

Compute the arithmetic mean of a numeric column:

```ts title="src/repositories/review.repo.ts"
import { Review } from "./review.model";

const avgRating = await Review.query()
  .where("productId", 42)
  .avg("rating");
// => 4.3
```

### min(field)

Return the minimum value of a column:

```ts title="src/repositories/product.repo.ts"
import { Product } from "./product.model";

const cheapest = await Product.query()
  .where("active", true)
  .min("price");
// => 4.99
```

### max(field)

Return the maximum value of a column:

```ts title="src/repositories/product.repo.ts"
import { Product } from "./product.model";

const mostExpensive = await Product.query()
  .where("active", true)
  .max("price");
// => 2499.00
```

### distinct(field)

Return an array of distinct values for a column:

```ts title="src/repositories/order.repo.ts"
import { Order } from "./order.model";

const regions = await Order.query().distinct<string>("region");
// => ["north", "south", "east", "west"]
```

### countDistinct(field)

Count the number of unique values of a column:

```ts title="src/repositories/order.repo.ts"
import { Order } from "./order.model";

const uniqueBuyers = await Order.query()
  .where("status", "paid")
  .countDistinct("userId");
// => 382
```

## 🔢 Increment & Decrement

Atomically update a numeric column on all rows matching the current filters.

### increment(field, amount?)

Increase a column by `amount` (defaults to 1). Returns the **new value** of the column on the first matched row.

```ts title="src/services/vote.service.ts"
import { Post } from "./post.model";

// Increment by 1
const newCount = await Post.query().whereId(99).increment("views");

// Increment by a custom amount
const newPoints = await Post.query().whereId(99).increment("points", 5);
```

### decrement(field, amount?)

Decrease a column by `amount` (defaults to 1). Returns the **new value**.

```ts title="src/services/inventory.service.ts"
import { Product } from "./product.model";

const remaining = await Product.query()
  .whereId(7)
  .decrement("stock", 3);
```

### incrementMany(field, amount?)

Increment a column across **all** matched rows. Returns the number of rows modified.

```ts title="src/jobs/daily-bonus.job.ts"
import { User } from "./user.model";

const modified = await User.query()
  .where("active", true)
  .incrementMany("dailyPoints", 10);

console.log(`Awarded bonus to ${modified} users`);
```

### decrementMany(field, amount?)

Decrement a column across **all** matched rows. Returns the number of rows modified.

```ts title="src/jobs/expire-credits.job.ts"
import { User } from "./user.model";

const modified = await User.query()
  .where("creditsExpireAt", "<", new Date())
  .decrementMany("credits");
```

## 🔎 Combining Scalar Aggregates with Filters

All scalar methods respect every `where*` clause added before them:

```ts title="src/services/analytics.service.ts"
import { Sale } from "./sale.model";

const [total, avg, max] = await Promise.all([
  Sale.query().where("year", 2025).where("region", "north").count(),
  Sale.query().where("year", 2025).where("region", "north").avg("amount"),
  Sale.query().where("year", 2025).where("region", "north").max("amount"),
]);
```
