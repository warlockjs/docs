# 🔃 Sorting

Control the order of query results with the QueryBuilder's ordering methods.

## orderBy

Sort by a single field, optionally specifying direction (`"asc"` or `"desc"`). Defaults to ascending.

```ts title="src/repositories/user.repo.ts"
import { User } from "./user.model";

// Ascending (default)
const users = await User.query().orderBy("name").get();

// Descending
const users = await User.query().orderBy("createdAt", "desc").get();
```

### Sort by multiple fields

Pass an array of `[field, direction]` tuples:

```ts
const users = await User.query()
  .orderBy([
    ["lastName", "asc"],
    ["firstName", "asc"],
  ])
  .get();
```

## orderByDesc

Shorthand for descending sort:

```ts
const users = await User.query().orderByDesc("createdAt").get();
```

## orderByRaw

Inject a raw driver expression for the ORDER BY clause:

```ts
const users = await User.query()
  .orderByRaw("LOWER(name) ASC")
  .get();
```

Pass bindings as the second argument when needed:

```ts
const users = await User.query()
  .orderByRaw("FIELD(status, ?) ASC", ["active"])
  .get();
```

## orderByRandom

Return a random subset of rows. The `limit` argument is required because fetching all rows in random order is expensive:

```ts
const randomUsers = await User.query().orderByRandom(10).get();
```

## latest

Sort descending by a date column. Defaults to `createdAt`:

```ts
// By createdAt (default)
const posts = await Post.query().latest().get();

// By a different column
const posts = await Post.query().latest("publishedAt").get();
```

:::info
`latest()` is a terminal method — it returns `Promise<T[]>` directly without needing a `.get()` call.
:::

## oldest

Sort ascending by a date column. Defaults to `createdAt`:

```ts
// Oldest first
const posts = await Post.query().oldest().get();

// By a different column
const posts = await Post.query().oldest("publishedAt").get();
```

## Chaining Multiple Sorts

All ordering methods are chainable and compose in the order they are called:

```ts title="src/repositories/product.repo.ts"
import { Product } from "./product.model";

const products = await Product.query()
  .where("active", true)
  .orderBy("category", "asc")
  .orderBy("price", "desc")
  .get();
```
