# Introduction

**Warlock.js** is a TypeScript-first Node.js framework for building fast, scalable APIs. Built on [Fastify](https://fastify.dev/) and [MongoDB](https://www.mongodb.com/), it provides everything you need to create production-ready backends.

## Why Warlock.js?

Warlock.js was designed with frontend developers in mind. You don't need deep backend expertise to build robust APIs:

- **Fast setup** - Create a working API in under 15 minutes
- **Convention over configuration** - Sensible defaults that just work
- **TypeScript-first** - Full type safety and excellent IDE support
- **Module-based** - Organize code by feature, not by type

## Quick Example

Here's a complete API endpoint in Warlock.js:

```ts title="src/app/users/routes.ts"
import { router, Request, Response } from "@warlock.js/core";
import { User } from "./models/user";

router.get("/users", async (request: Request, response: Response) => {
  const users = await User.list();
  return response.success({ users });
});
```

## Key Features

### Core Framework
- Hot Module Reload (HMR) for fast development
- Event-driven architecture
- Automatic route and module loading
- Route-based middleware with groups

### Data Layer ([Cascade](./../../cascade/getting-started/introduction.mdx))
- Full MongoDB support with powerful ORM
- Model relationships and syncing
- Migrations and schema management

### API Development
- RESTful API support with a single base class
- Request validation with [Seal](./../../seal/getting-started/introduction.mdx)
- Response transformation with [Outputs](./../http/output.mdx)
- Postman collection generator

### Authentication & Security
- JWT-based authentication
- Role-based access control
- Guest user support

### Additional Features
- File uploads with image compression
- AWS S3 integration
- Email with React components
- [Redis caching](./../../cache/introduction.mdx)
- Internationalization (i18n)
