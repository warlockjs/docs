## Pre-requisites

Warlock.js requires Node.js version <span className="text-primary">18</span> or higher. You can download and install Node.js from the official website [nodejs.org](https://nodejs.org/).

Make sure to check [MongoDB Requirements](./../../cascade/getting-started/installation.mdx) as well.

## Installation

To start a new Warlock.js Project, run the following command:

<Tabs>
  <TabItem value="Yarn" label="Yarn">

```bash
yarn create warlock
```

  </TabItem>
  <TabItem value="NPM" label="NPM">

```bash
npx create-warlock
```

  </TabItem>
  <TabItem value="Pnpm" label="Pnpm">

```bash
pnpm create warlock
```

  </TabItem>
</Tabs>

Follow the installation instructions and you're good to go!

## VSCode Extension

Warlock.js modules could be generated using [Generator Z](https://marketplace.visualstudio.com/items?itemName=warlockjs.generator-z) VSCode extension. You can install the extension from the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=hassanzohdy.gnz).

## Run the project

Open the project directory then run the following command:

<Tabs>
  <TabItem value="Yarn" label="Yarn">

```bash
yarn start
```

  </TabItem>
  <TabItem value="NPM" label="NPM">

```bash
npm start
```

  </TabItem>
  <TabItem value="Pnpm" label="Pnpm">

```bash
pnpm start
```

  </TabItem>
</Tabs>
