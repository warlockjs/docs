# Introduction

Warlock is shipped with MongoDB as default database, which is now the only supported database engine.

MongoDB in `Warlock` is powered by [Cascade](./../../cascade/getting-started/introduction) package, it's a wrapper around the official MongoDB driver with tremendous features and incredible performance.

:::info
In the next major release there will be support for other databases like MySQL, Postgres, and SQLite.
:::
